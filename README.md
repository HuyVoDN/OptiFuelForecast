## Set Up
 Clone the repo
```
git clone https://github.com/HuyVoDN/OptiFuelForecast.git
```

Install the dependencies for each folder
```
cd back-end
npm i
```
```
cd front-end
npm i
```
## Running the project
- Running the server: 
```
cd back-end
npm start
```
- Running the client: 
```
cd front-end
npm run dev
```
